## Description: <br>
UI/UX Pro Max provides UI/UX design intelligence and implementation guidance for polished interfaces, including layout concepts, UX flows, design-system tokens, component guidance, accessibility coverage, and frontend code recommendations. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers, designers, and product teams use this skill to plan, critique, and implement frontend UI/UX work. It supports interface layouts, user flows, design-system recommendations, accessibility checks, and concrete code-oriented guidance across common web and app stacks. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Optional persistence commands can create or overwrite design-system Markdown files in the workspace when flags such as --persist, --output-dir, project names, or page names are used. <br>
Mitigation: Review the command options and output directory before running persistence commands, and run them in the intended workspace or a version-controlled branch. <br>
Risk: UI/UX recommendations may be incomplete or unsuitable for a specific product, brand, accessibility target, or implementation stack. <br>
Mitigation: Review generated recommendations against product requirements, accessibility criteria, and the target frontend stack before implementation. <br>
